# @ro-claude-kit/agent-patterns

Production-grade agent patterns for Claude — implemented in Python with a TypeScript wrapper.

Patterns shipped (Week 1):
- ReAct (with reflection + tool retry + iteration cap)
- Planner-Executor (with checkpoint/resume + plan revision)
- Multi-Agent Supervisor (orchestrator + typed sub-agent state)
- Reflexion (act → reflect → retry-with-self-critique)

Status: skeleton only. Implementations land in Step 6+.
